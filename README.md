 🌐 Personal Portfolio Website

A clean and responsive   Personal Portfolio Website   built using   HTML, CSS, and JavaScript  .
This project showcases my skills, projects, education, and work experience in a simple yet modern design.


 🚀 Features

  🌙 Dark Mode Toggle (with localStorage support)
  📱 Fully Responsive Design (Mobile + Desktop)
  🎨 Clean and Minimal UI
  📂 Multi-page Structure
  ⭐ Smooth Hover Effects
  🧠 Beginner-friendly structure


 📁 Project Structure

```
├── index.html
├── projects.html
├── articles.html
├── contact.html
├── style.css
└── README.md
```


 🛠️ Technologies Used

  HTML5
  CSS3 (Flexbox + Responsive Design)
  JavaScript (DOM Manipulation)
  Google Fonts (MedievalSharp)


 ⚙️ How to Run

1. Download or clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Open the project folder

3. Run `index.html` in your browser


 🌙 Dark Mode

  Toggle switch available in navbar
  Saves user preference using   localStorage  


 📌 Pages Included

    Home   → Introduction
    Projects   → List of projects
    Articles   → Learning/blog section
    Contact   → Contact form
